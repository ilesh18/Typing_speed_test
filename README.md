<h1>Typing Speed Test</h1> <h2>Overview of the Project</h2> <p>This is a command-line Python application designed to test and improve typing speed and accuracy. Users can participate in standard typing tests or timed challenges, track their progress over time, unlock achievements, and visualize improvements through graphs. All data is stored in a MySQL database for persistence across sessions. The app is ideal for individuals looking to practice typing skills, with features inspired by popular typing test tools.</p> <h2>Features</h2> <ul> <li><strong>Typing Test Mode</strong>: Complete 5 random sentences, calculate average WPM (Words Per Minute), track errors, and save results.</li> <li><strong>Typing Challenge Mode</strong>: Choose difficulty levels (Easy: 60s, Medium: 45s, Hard: 30s) and type within a time limit.</li> <li><strong>Achievements System</strong>: Unlock badges for milestones like reaching 60+ WPM, perfect accuracy, or completing multiple tests.</li> <li><strong>Progress Tracking</strong>: View historical results and plot the last 5 attempts using Matplotlib.</li> <li><strong>Database Persistence</strong>: Store user results and achievements in MySQL for long-term tracking.</li> <li><strong>Error Analysis</strong>: Uses sequence matching to detect and report incorrect words.</li> <li><strong>User-Friendly Menu</strong>: Simple CLI interface with options to start tests, view data, or exit.</li> </ul> <h2>Technologies/Tools Used</h2> <ul> <li><strong>Programming Language</strong>: Python 3.7+</li> <li><strong>Database</strong>: MySQL (for data storage and retrieval)</li> <li><strong>Libraries</strong>: <ul> <li><code>mysql-connector-python</code>: For MySQL database connections.</li> <li><code>matplotlib</code>: For generating progress plots.</li> <li><code>difflib</code>: For comparing user input to target sentences (built-in Python library).</li> <li><code>getpass</code>, <code>random</code>, <code>time</code>: Built-in Python modules for password input, randomization, and timing.</li> </ul> </li> <li><strong>Other Tools</strong>: Command-line interface (no GUI required, but Matplotlib assumes a display for plots).</li> </ul> <h2>Steps to Install & Run the Project</h2> <h3>Prerequisites</h3> <ul> <li>Python 3.7 or higher installed on your system.</li> <li>MySQL Server installed and running (e.g., via XAMPP, WAMP, or native installation).</li> <li>A MySQL user with privileges to create databases (default assumed: 'root').</li> </ul> <h3>Installation Steps</h3> <ol> <li><strong>Clone or Download the Code</strong>: install --upgrade pip</code>.</li> </ul> </li> <li><strong>Set Up MySQL</strong>: <ul> <li>Start your MySQL server.</li> <li>Ensure the 'root' user exists and has database creation permissions. If needed, create a new user via MySQL CLI:</li> </ul> <pre><code>CREATE USER 'root'@'localhost' IDENTIFIED BY 'your_password'; GRANT ALL PRIVILEGES ON *.* TO 'root'@'localhost'; FLUSH PRIVILEGES;</code></pre> </li> <li><strong>Run the Script</strong>: <ul> <li>Navigate to the directory containing <code>typing_speed_test.py</code>.</li> <li>Execute the script:</li> </ul> <pre><code>python typing_speed_test.py</code></pre> <ul> <li>On first run, enter your MySQL root password when prompted.
